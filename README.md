# Telegram Bot for image converting

## Prerequisites

- Go 1.23 or later

## 

1. **Setup:**

- Get Bot token from @BotFather
- Create .env and edit the missed parameters
- Build and start **blackandwhite** microservice in **cmd/blackandwhite** 

2. **Build and run locally:**

    ```sh
    make dev
    ```
